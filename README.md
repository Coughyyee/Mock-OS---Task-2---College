# MOCK OS Task 2 for College
This is the code for the Task 2 section of the mock exam.
This solution is for a Health Advisory Group where users can view common conditions and get advice based on them, view weather and airquality data in specified locations with advice based on the data, 
booking of risk assessments and a personal health tracking tool. This application includes secure Authentication and Authorization for certain endpoints and requests. Users are allowed to sign up and login to their accounts
where they can mange their own risk assessment bookings and access the personal health tracking tool. Admin functionality is included - admin accounts can login and view all risk assessment bookings where they can mark
them as completed.

### Stack
- PHP
- SQL
- HTML
- CSS
  - Tailwindcss
  - DaisyUI

### Folder structure
- **app**: backend logic of the application
- **app/db**: database logic
- **app/models**: data transfer objects / models
- **app/service**: controller classes (main file for managing requests, data, etc.)

- **assets**: images and JSON sample data

- **public**: all pages to be viewed.

### Project Completion Info
This project took 1.5 weeks to build therefore most logic is functional inside of the application but there are some areas that could be improved / developed.
